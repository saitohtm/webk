Recruit.UI.*: a part of Recruit Web Service UI Library

TOP > 共通系

概要

リクルート WEBサービス UI Library の中でも、すべてのWEBサービス共通で使えるライブラリをここにまとめています。

* ドコイク？やフロムエー・ナビなど、独自でAPI公開をしている一部のWebサービスには対応していません。リクルートWebサービス 公式サイトの左メニューにて、青色帯で表示されているものが対応しているWebサービスになります。
